# Tabify

**Tabify** は、あらゆる Windows アプリケーションを一つのウィンドウに統合し、タブ形式で管理できるようにするユーティリティツールです。

Stardock Groupy にインスパイアされ、より軽快でモダンな UX を目指して開発されています。

## ✨ 特徴

- **ドラッグ＆ドロップ統合**: 既存のウィンドウを Tabify のタブバーにドロップするだけで、即座にタブとして取り込まれます。
- **ブラウザライクな操作感**: Chrome のようなタブの並び替え、タブ間のスムーズな切り替えが可能です。
- **タイトルバーレス・デザイン**: アプリの表示領域を最大限に確保するため、タイトルバーを廃止し、タブバー自体でウィンドウの移動が可能です。
- **リサイズ・スナップ対応**: Windows の標準的なリサイズ操作やスナップ機能とも親和性が高く、快適に動作します。
- **ハイライト演出**: ウィンドウをドロップする際、統合可能なエリアを視覚的にガイドします。

## 🚀 使い方

1. `Tabify.exe` を起動します。
2. 統合したいアプリケーション（秀丸、VSCode、エクスプローラーなど）のタイトルバーを掴みます。
3. Tabify のタブバー（上部のグレーのエリア）にドロップします。
4. ウィンドウがタブとして取り込まれます。

※ タブを右クリックすることで「切り離し (Detach)」が可能です。
※ タブの「✕」ボタンを押すと、そのアプリケーションを終了させることができます。

## 🛠 開発環境

- **言語**: C#
- **フレームワーク**: .NET 8.0 (WPF)
- **API**: Win32 API (User32, Kernel32) を活用してウィンドウ外観を制御
