# Tabify

**Tabify** は、あらゆる Windows アプリケーションを一つのウィンドウに統合し、タブ形式で一元管理できるようにするユーティリティツールです。

Stardock Groupy にインスパイアされ、より軽快でモダンな UX を目指して開発されています。

## ✨ 特徴

- **ドラッグ＆ドロップ統合**: 既存のウィンドウを Tabify のタブバーにドロップするだけで、即座にタブとして取り込まれます。
- **ブラウザライクな操作感**: Google Chrome のような流れるようなタブの並び替えや、**タブをウィンドウ外へドラッグしての別ウィンドウへの切り離し・結合**が可能です。
- **自由自在なタブ配置**: タブバーの位置設定（上・下・左・右）に柔軟に対応し、スクロールにも対応します。
- **ダークモード/ライトモード連動**: Windows 11 のシステムテーマ（ダーク/ライト）設定をリアルタイムに検知し、自動で最適な UI カラーを適用します。
- **エクスプローラーネイティブ対応**: 標準の Windows エクスプローラー（CabinetWClass）も無理なくタブ化・統合が可能です。
- **タイトルバーレス＆直感的操作**: タイトルバーをダブルクリックしての「最大化/元に戻す」トグル操作に対応。表示中の全ウィンドウを1つにまとめる「一括集約 (Aggregate All)」ボタンも搭載しています。

## 🚀 使い方

1. `Tabify.exe` を起動します。
2. 統合したいアプリケーション（ブラウザ、VSCode、エクスプローラーなど）のタイトルバーを掴みます。
3. Tabify の「ここにウィンドウをドロップして追加」エリアにドロップします。
4. ウィンドウがタブとして取り込まれます。

**便利な操作タブ:**
- **タブの切り離し**: タブをウィンドウ外へドラッグ＆ドロップするか、右クリックから「切り離し (Detach)」を選択します。
- **タブの結合**: 切り離したタブを、別の Tabify ウィンドウにドロップすると結合されます。
- **タブを閉じる**: タブ内の「✕」ボタンを押すと、そのアプリケーション自体が終了します。
- **最大化・最小化**: タイトルバーのダブルクリックで最大化と元のサイズの切り替えが可能です。
- **表示設定**: 設定 (⚙) ボタンから、タブの位置（上下左右）の変更や現在のバージョン確認が行えます。

## 🛠 開発・ビルド環境

- **言語**: C#
- **フレームワーク**: .NET 8.0 (WPF)
- **API**: Win32 API (User32, Kernel32, DwmApi) を活用した高度なウィンドウフックとキャプチャ技術（RenderTargetBitmap）
